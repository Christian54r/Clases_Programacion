Understanding HTML Tags

Important HTML Tags

  • <html>: The root element of an HTML document.
  • <head>: Contains metadata and links to external resources like stylesheets.
  • <title>: Sets the title of the webpage (displayed on the browser tab).
  • <body>: Contains the content of the webpage.
  • <h1> to <h6>: Headings, with <h1> being the most important.
  • <p>: Defines a paragraph.
  • <a>: Creates hyperlinks.
  • <img>: Embeds images.
  • <ul> and <ol>: Unordered and ordered lists.
  • <div>: Defines a division or section in the document.
